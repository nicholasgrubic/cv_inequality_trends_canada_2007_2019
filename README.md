# cv_inequality_trends_canada_2007_2019
Analysis codes for "Income Inequalities in Cardiovascular Health Among Children and Adolescents, Young Adults, and Middle-Aged Adults in Canada: A Nationwide Trend Analysis, 2007-2019" by Nicholas Grubic, Sarah Carsley, Arjumand Siddiqi, Cristian Carmeli, and Katerina Maximova.

- _DescriptiveHealthInequalityAnalyses_05052026.R_ - Main descriptive and health inequality analyses
- _ConditionSpecificDescriptiveAnalyses_05052026.R_ - Descriptive analyses for individual cardiovascular health conditions
- _SensitivityAnalyses_05052026.R_ - Sensitivity analyses
- _SupplementalAnalyses_05052026.R_ - Supplemental analyses
